#Phaer.io and Typescript boilerpate
Boilerplate code for phaer.io and Typescript based on http://www.photonstorm.com/phaser/advanced-phaser-and-typescript-projects

#Compile
tsc --out game.js boot.ts game.ts preloader.ts mainmenu.ts player.ts level1.ts app.ts
If you want that the tsc is watching your *ts files then you can just put -w into the command
